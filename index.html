2. Cheatsheet

General

"cd /pfad" = change directory

"cd /" = change to root

"git clone" = repository klonen

"git add *" = adds all file

"git commit -m "message" " = commit with message

"git push" = push to repository

Branches

"git branch _branchname_" = create branch

"git checkout _branchname_" = change branch

"git merge _branchname_" = merge branches
